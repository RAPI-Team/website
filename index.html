üéØ Getting Started

RAPI provides simple, static classes to interact with Roblox version information.

C# using RAPI; // Get player version for current OS string version = PlayerVersion.Current; // Get Studio version string studioVer = StudioVersion.Version; // Check local installation string installed = LocalVersion.GetInstalled(); // Compare versions var result = await LocalVersion.CompareAsync();

üìö Classes Overview

APIInfo

Contains information about the RAPI library itself.

APIInfo.Version - Returns the API version (e.g., "1.0.0")
APIInfo.Creator - Returns "RAPI Team"

OS

Detects the current operating system.

OS.Windows - Returns true if running on Windows
OS.Mac - Returns true if running on macOS
OS.Linux - Returns true if running on Linux
OS.Name - Returns OS name as string ("Windows", "Mac", "Linux", "Unknown")

PlayerVersion

Retrieves Roblox Player version information from official APIs.

PlayerVersion.Windows - Gets Windows Player version
PlayerVersion.Mac - Gets Mac Player version
PlayerVersion.Linux - Gets Linux Player version
PlayerVersion.Current - Gets version for current OS
PlayerVersion.ClearCache() - Clears cached versions
‚ö° Caching: Results are cached for 5 minutes to improve performance.

StudioVersion

Retrieves Roblox Studio version information.

StudioVersion.Version - Gets current Studio version
StudioVersion.ClearCache() - Clears cached version

LocalVersion

Checks locally installed Roblox version and compares with online version.

LocalVersion.GetInstalled() - Gets locally installed version
await LocalVersion.GetOnlineVersionAsync() - Gets online deployment version
await LocalVersion.CompareAsync() - Compares local vs online versions

VersionComparisonResult

Result object returned by LocalVersion.CompareAsync().

OnlineVersion - The online version string
LocalVersion - The local version string
Status - VersionStatus enum (UpToDate, UpdateAvailable, Error)
Message - User-friendly message describing the result

üí° Usage Examples

WinForms Example

C# using RAPI; public partial class Form1 : Form { public Form1() { InitializeComponent(); LoadVersions(); } private void LoadVersions() { // Display versions in labels lblPlayerVersion.Text = PlayerVersion.Current; lblStudioVersion.Text = StudioVersion.Version; lblLocalVersion.Text = LocalVersion.GetInstalled(); lblOS.Text = OS.Name; } private async void btnCompare_Click(object sender, EventArgs e) { var result = await LocalVersion.CompareAsync(); if (result.Status == VersionStatus.UpToDate) { MessageBox.Show(result.Message, "Up to Date", MessageBoxButtons.OK, MessageBoxIcon.Information); } else if (result.Status == VersionStatus.UpdateAvailable) { MessageBox.Show(result.Message, "Update Available", MessageBoxButtons.OK, MessageBoxIcon.Warning); } else { MessageBox.Show(result.Message, "Error", MessageBoxButtons.OK, MessageBoxIcon.Error); } } }

WPF Example

C# using RAPI; public partial class MainWindow : Window { public MainWindow() { InitializeComponent(); LoadVersions(); } private void LoadVersions() { txtPlayerVersion.Text = PlayerVersion.Current; txtStudioVersion.Text = StudioVersion.Version; txtLocalVersion.Text = LocalVersion.GetInstalled(); txtOS.Text = OS.Name; } private async void CompareButton_Click(object sender, RoutedEventArgs e) { txtStatus.Text = "Checking..."; var result = await LocalVersion.CompareAsync(); txtStatus.Text = result.Message; txtOnlineVersion.Text = result.OnlineVersion; txtLocalVersion.Text = result.LocalVersion; } }

Console Example

C# using RAPI; class Program { static async Task Main(string[] args) { Console.WriteLine($"=== RAPI v{APIInfo.Version} ==="); Console.WriteLine($"Operating System: {OS.Name}"); Console.WriteLine($"Player Version: {PlayerVersion.Current}"); Console.WriteLine($"Studio Version: {StudioVersion.Version}"); Console.WriteLine($"Local Install: {LocalVersion.GetInstalled()}"); Console.WriteLine("\nComparing versions..."); var result = await LocalVersion.CompareAsync(); Console.WriteLine(result.Message); } }

üìñ API Reference

Return Types

All synchronous properties return string
Async methods return Task<string> or Task<VersionComparisonResult>
Properties return "unknown" on error (except LocalVersion which returns descriptive error messages)

Caching Behavior

üïê Cache Duration: 5 minutes

PlayerVersion and StudioVersion cache results for 5 minutes to reduce API calls. Use ClearCache() methods to force a refresh.

Error Handling

‚úÖ Best Practice:
string version = PlayerVersion.Current; if (version == "unknown") { // Handle error case MessageBox.Show("Failed to fetch version"); } else { // Use version lblVersion.Text = version; }

Thread Safety

All RAPI classes are thread-safe. HttpClient instances are reused and properly configured.
